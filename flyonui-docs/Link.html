Tailwind CSS Link

The link component facilitates setting hyperlinks from inline text items, buttons, or cards to navigate between pages or external websites.

Class Name
Type
Description
link Component Adds underline to a text.
link-hover Modifier Only show underline on hover.
link-animated Modifier Show underline on hover with animation.
link-neutral Modifier Applies’neutral’ color to link.
link-primary Modifier Applies’primary’ color to link.
link-secondary Modifier Applies’secondary’ color to link.
link-accent Modifier Applies’accent’ color to link.
link-success Modifier Applies’success’ color to link.
link-info Modifier Applies’info’ color to link.
link-warning Modifier Applies’warning’ color to link.
link-error Modifier Applies’error’ color to link.

Variants

Default

The link component class serves as the standard class for the basic anchor tag.

Default link 
<a href="#" class="link">Default link</a>

Underline on hover

Apply the link-hover modifier class to make links underlined when hovered over.

Underline when hovered over. 
<a href="#" class="link link-hover">Underline when hovered over.</a>

Animated underline

Apply the link-animated modifier class to make links underlined with animation on hovered over.

Animated underline when hovered over. 
<a href="#" class="link link-animated"> Animated underline when hovered over.</a>
The modifier classes `link-animated` and `link-hover` are also applicable to semantic links.

Colors

Semantic links

Utilize the link-{semantic-color} modifier class to assign specific semantic colors to links.
