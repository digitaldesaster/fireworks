Tailwind CSS Navbar

The navbar component provides versatile options for displaying navigation links at the top of your page, with various layouts, sizes, and dropdowns.

Class Name
Type
Description
navbar Component Base class for navbar container.
navbar-start Component Child element, fills 50% of width to be on start.
navbar-center Component Child element, fills remaining space to be on center.
navbar-end Component Child element, fills 50% of width to be on end.
