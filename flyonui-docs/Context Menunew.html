Tailwind CSS Context Menu

Enhance interactivity and user experience on your site by implementing a context menu that offers users relevant actions and options through a right-click.

Require FlyonUI JS
Note that this component requires the use of the third-party Popper.js and FlyonUI Dropdown plugin, else you can skip this message if you are already using FlyonUI as a package.

Basic example

Default

Utilize [--trigger:contextmenu] to activate the context menu.

Right click
