tables

Tailwind CSS Table

Tables are designed to present data in a structured tabular format, making it easy to read and analyze information efficiently.

Class Name
Type
Description
table Component Base class for the table component.
table-borderless Modifier Removes border from table.
table-striped Modifier For table to show striped rows.
table-striped-columns Modifier For table to show striped columns.
table-pin-rows Modifier For table to make all the rows inside thead and tfoot pinned.
table-pin-cols Modifier For table to make all the th columns pinned.
table-xs Responsive Table with extra small size.
table-sm Responsive Table with small size.
table-lg Responsive Table with large size.
hover Utility For table rows hover effect.
